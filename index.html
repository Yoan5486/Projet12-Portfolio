Je m'appelle Yoan COCQUEBERT, j'ai 21 ans, et je suis diplômé en tant qu'Intégrateur Web.

Après un baccalauréat professionnel en Systèmes Numériques,
j'ai suivi la formation "Intégrateur Web" proposée par OpenClassrooms.

Je maîtrise le HTML, CSS, animations CSS, JavaScript, JSX, React, Redux, les API, GitHub, SEO, le débogage, les tableaux Kanban et les outils de veille technologique.

Je vous présente ici deux projets réalisés dans le cadre de ma formation :

Découverte de React

Projet nommé par OpenClassrooms : "Créez une application web de location immobilière avec React"

Dans ce projet, j'ai réalisé un carrousel, des boucles JavaScript, des accordéons (collapses), et organisé le code en composants réutilisables.

Ce projet m'a permis de bien comprendre React, d’utiliser ses hooks comme useState et useNavigate, et de développer du JSX avancé.

Animations CSS

Projet nommé par OpenClassrooms : "Améliorez l'interface d'un site mobile avec des animations CSS"

Dans ce projet, j’ai intégré une animation CSS de chargement (loader), ainsi que des animations sur les logos et le texte apparaissant progressivement (comme sur ce site).

Ce travail m'a permis de développer ma créativité en matière de design, de mieux maîtriser les animations via les keyframes et les transitions CSS.

Les compétences acquises durant ma formation me permettront de faciliter mon insertion professionnelle, notamment en tant qu’Intégrateur Web. J’ai développé des savoir-faire concrets, maîtrisé des outils professionnels du secteur, et mieux compris les attentes du monde professionnel.
